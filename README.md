# hospitality-project
repo for work done on power bi for a hospitality project.
Hospitality Industry Data Analysis - Power BI
🏨 Project Overview
This project involves the creation of an interactive Power BI dashboard designed to provide actionable insights for a hospitality business (Hotels/Resorts). The goal was to analyze key performance indicators (KPIs) to help management improve revenue, occupancy rates, and customer satisfaction.

🛠️ Technical Toolkit
Tool: Power BI Desktop

Data Cleaning: Power Query (M-Query)

Data Modeling: Star Schema (Fact and Dimension tables)

Calculations: DAX (Data Analysis Expressions) for complex measures.

📊 Key Metrics & DAX Measures
I have included standard hospitality metrics here. Please update these with the actual measures you created in your file:

RevPAR (Revenue Per Available Room): Total Revenue / Total Rooms Available

ADR (Average Daily Rate): Total Room Revenue / Number of Rooms Sold

Occupancy %: (Rooms Sold / Rooms Available) * 100

Cancellation Rate: Analyzing the impact of booking cancellations on projected revenue.

🏗️ Data Model
The analysis is built on a Star Schema to optimize performance:

Fact Table: Fact_Bookings (Revenue, Date, Room ID, etc.)

Dimension Tables: Dim_Hotels, Dim_Rooms, Dim_Date.

💡 Business Insights
Booking Channels: Identified which platforms (Direct, OTA, Agent) yield the highest profit margins.

Trend Analysis: Detected seasonal peaks to optimize pricing strategies (ADR).

Customer Behavior: Analyzed the relationship between room types and cancellation frequency.
